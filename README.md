# Neural Networks and Autoencoders in Python

*Advanced undergraduate machine learning coursework project (University College Dublin)*

## Overview
This repository contains a set of neural network implementations developed as part of advanced undergraduate coursework in the Maths of Machine Learning module. The project focuses on understanding the foundations of neural networks through both conceptual analysis and practical implementation, with particular emphasis on autoencoders, representation learning, and supervised regression models.

The work combines theoretical reasoning with hands-on experimentation using Python and TensorFlow/Keras, exploring how network architecture, activation functions, and training choices affect model performance and generalisation.

- Investigated convergence, generalisation, and sources of model error.
- Visualised training dynamics, reconstructions, predictions, and residuals.
- Structured the work in a clear, reproducible Jupyter notebook with explanatory Markdown.

## Technical focus

- **Frameworks:** TensorFlow, Keras
- **Machine learning:** neural networks, autoencoders, representation learning
- **Concepts:** optimisation, loss functions, overfitting, generalisation, model evaluation
